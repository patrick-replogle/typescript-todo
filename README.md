# Todo App

Deployed web app can be found here: https://typescript-todo-web-app.netlify.app/

## Tech Stack

- TypeScript
- React
- Sass
- Material UI
- Netlify

## Project Overview

A simple todo list web app to help you keep track of tasks that need to get done.
